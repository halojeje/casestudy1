Tujuan Pembelajaran:
Mahasiswa memahami dan mengimplementasikan berbagai tipe data dasar di Dart (int, double, String, bool, List, Map)


Mahasiswa mampu menggunakan comment (//, /* */, ///) untuk menjelaskan bagian kode



Studi Kasus: Digital Biodata Anggota Klub
Sebuah klub buku digital sedang mengembangkan sistem sederhana untuk menyimpan biodata anggotanya.
Anda diminta untuk membuat program berbasis Dart yang menyimpan dan menampilkan data anggota, lengkap dengan dokumentasi menggunakan comment.

Instruksi Tugas:
Buat variabel untuk menyimpan informasi anggota berikut:


Nama (String)


Umur (int)


Tinggi badan (double)


Status aktif (bool)


Daftar buku favorit (List)


Informasi tambahan disimpan dalam bentuk Map (contoh: alamat, profesi)


Gunakan:


// untuk komentar satu baris


/* ... */ untuk penjelasan multi-baris
